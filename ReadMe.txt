capstone project

This is a web application for parking reservation system.

The front end, including searching, reservation, payment and navigation parts

The back end is a relational database to save event info, user info, reservation info etc.

Implement PayPal API to finalize payment and reservation, and Google Maps API as search and navigation function

Originally idea coming from football games hold in Mizzou, which People have to come very early to get a parking space
